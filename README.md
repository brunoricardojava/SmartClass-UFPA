# SmartClass-UFPA
O SmartClass é um projeto ousado que tem o objetivo de tornar as salas de aula um espaço mais inteligente.

Aproveitando de tecnologias OpenSource, pretendemos aplicar os conhecimentos adquiridos no curso de engenharia da computação da UFPA, para tornar a sala de aula em um espaço mais inteligente, onde serviços estejam conectados, facilitando tanto o processo de aprendizado como abrindo uma nova possibilidade para sistemas educacionais mais modernos.

Temos o objetivo de moldar um sistema modelo de salas de aula, que possa ser usado nos mais diversos lugares ou que sirva de inspiração para novos models.
